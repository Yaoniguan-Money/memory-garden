# Memory Garden Documentation

Memory Garden is a **local-first, auditable memory layer for AI agents**. It gives your agent a structured memory lifecycle — from candidate observation through court judgment, growth, dream cycles, harvesting, and runtime orchestration — without depending on cloud services, vector databases, or external LLM providers.
